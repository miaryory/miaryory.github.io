Magento

Déscription

Réalisation d'un plugin personnalisé reliant un magasin Magento à la plateforme LogiSnap.
LogiSnap est une compagnie danoise développant un outil qui permet aux entreprises de mieux gérer leur logistique.

Fonctionnalités

  • Authentication
  • Token verification
  • Create, Update order
  • Enable, Disable Shipping methods

Répertoire Github

https://github.com/miaryory/magento

Outils

PHP, Magento, REST API, MySQL, Postman

Le plugin installé dans la partie Admin de Magento
